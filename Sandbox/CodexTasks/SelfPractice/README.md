# Self Practice

Independent Java practice tasks.

Use this folder for exercises you try without AI first. The goal is not to write perfect code immediately, but to practice thinking, debugging, and improving your own solution.

## Practice Areas

- `InputValidation` - console input, validation, loops, arrays, and conditions.

## Completed Tasks

- `InputValidation/task1` - Exam Average.
- `InputValidation/task2` - Temperature Log.

## Suggested Workflow

1. Read the task.
2. Write a rough solution by yourself.
3. Run it with normal and bad input.
4. Note what failed.
5. Ask for review only after you have a working or partially working version.

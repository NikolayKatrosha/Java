# Hillel

Course work and refreshed Java exercises from Hillel School.

## Folders

- `JavaElementary` - fundamentals, syntax, control flow, methods, arrays, and object-oriented basics.
- `JavaPro` - deeper Java topics, libraries, tooling, and more advanced assignments.

Each task should live in a focused folder with its source code and, when useful, a short README describing the assignment and solution approach.

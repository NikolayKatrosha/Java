# Hometask 1

Basic Java practice tasks from the Hillel Java Elementary course.

## Topics

- Variables and primitive types
- Console output
- Basic arithmetic
- `Math` utility methods
- Simple helper methods

## Tasks

- `Task1` - swap two variables.
- `Task2` - print geographic coordinates.
- `Task3` - calculate a mathematical expression.
- `Task4` - calculate snowman weight using sphere volume and density.
